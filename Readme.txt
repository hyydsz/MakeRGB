首先填写 宽度高度（不要太大）设备名称，显示名称，品牌
然后设备灯珠区会自动绘制表格
画完灯珠之后 点击生成 然后选择保存位置
接下来会再次弹出一个窗口 选择你的图片(可以提前把图片拖入到软件) 如果没有图片可以选择取消

将生成的.json的文件复制到我的电脑【文档】【W...FX】路径下的【Components】 内 并重新启动 SignalRGB！
在”设备“中选择所需的控制器并单击-它连接到的频道下方的图标。在“品牌”下拉菜单中 选择你自己的品牌

删减灯珠时 需要按照从大到小才能删减灯珠 可以按住鼠标左键拖拽灯珠
